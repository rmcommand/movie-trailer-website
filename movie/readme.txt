Moodey's Movie Website Python Script(R) version 1.2 04/03/2015

General Usage Notes
- Unzip the contents into the same directory

-Ensure that Python is installed on your computer. Linux will have the interpreter installed by default
but on Windows or Mac, use IDLE.

- Run the 'entertainment_center.py' script. Allowing for this script to run will make reference to
fresh_tomatoes.py script.
